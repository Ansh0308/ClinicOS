## Plain Text / Markdown Format

**ClinicOS — Clinic Management & Patient Engagement Platform** | *2025–2026*

- Engineered a full-stack clinic OS with **4 role-based portals** (Doctor, Reception, Patient, Admin) serving real-time queue management across all connected devices via **Socket.IO WebSockets**.
- Designed a live **token queue engine** with auto-calculated ETAs, dynamic queue reordering, pause/resume controls, and real-time position broadcasting — eliminating manual paper-token workflows.
- Integrated **Razorpay payment gateway** (UPI, Card, Netbanking) with webhook-based payment verification and automated **PDFKit** invoice generation, attached directly to **Nodemailer** confirmation emails.
- Built a structured **EHR (Electronic Health Records)** system for doctors — capturing vitals, complaints, diagnoses, and multi-drug prescriptions with full patient visit-history search.
- Implemented **JWT + bcrypt** authentication with role-based access control (RBAC) across 5 user roles, and an OTP patient portal for self-service queue tracking and bill payment.

**Technologies:** React 19, Vite, Tailwind CSS, Socket.IO, Node.js, Express 5, MySQL 8, Sequelize ORM, Razorpay, PDFKit, Nodemailer, WhatsApp Cloud API, JWT, bcryptjs, Recharts, Zod
